Detailed Description

The PPM UI.

Previously used the JProgressBar as level indicator. That one had problems [100]. Replaced it with a steelseries-swing gauge LinearBargraph component (https://github.com/HanSolo/SteelSeries-Swing) - which is very responsive.

Author
Peter Hoppe
