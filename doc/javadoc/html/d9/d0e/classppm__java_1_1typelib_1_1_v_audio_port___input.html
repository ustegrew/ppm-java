Detailed Description

Base class for an input audio port.


An input port is always associated with a hosting audio processor. It's connected with another output port from where it receives its data.

To ensure type safety the connection mechanism uses the visitor pattern for the connection to an output port. This class provides the prototype for the _Accept(VAudioPort_Output) method.
