Detailed Description

The Invalid data policy.

Determines what will be returned when the consumer calls TAtomicBuffer#Get() and we have no data to offer - either because nothing has been set since the last call to TAtomicBuffer#Get(), or because just in this moment the producer is setting a new sample chunk so we have a contention. To be supplied as argument to the constructor.

kReturnEmpty
(default). Return an empty sample chunk.
kReturnNull
Return null. This may be interesting for a consumer which wants to call TAtomicBuffer#Get() inside an endless loop as long as it returns null.
