Detailed Description

Central application point.

Contains the API to set up a ppm-java signal processing network and to run it. Note that every object we create is stored in the global registry and identifiable by a unique ID (opaque handle). This simplifies the syntax for the setup of the signal network enormously.

Author
Peter Hoppe
See Also
ppm_java.backend.boot.TSetup The service class that contains the signal processing network setup.
