Detailed Description

Base class for an output audio port.

An output port is always associated with a hosting audio processor. It's connected with another VAudioPort_Input to where it sends its data. Every time this port sends data it will trigger a processing cycle in the audio processor associated with the connected input.

To ensure type safety the connection mechanism uses the visitor pattern for the connection to an input port. This class is the visitor base class and provides the prototypes for the _Visit(...) methods.
