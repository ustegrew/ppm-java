25 /**
26  * Base class for an input port that receives sample chunks. This is a
27  * {@linkplain TAtomicBuffer buffered} port, i.e. sample chunks are
28  * received and stored in a {@linkplain TAtomicBuffer buffer}.

30  * on time.
