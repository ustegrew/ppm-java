Detailed Description

Loads the JJack driver, connects to a running instance of JackD and prints the peak of every incoming frame to stdout.

Note - this program never finishes by itself unless terminated from a thread separate from the main thread.

The process callback.

Called by the Jack server via the JJack bridge. Time critical! If we get stuck here, we also freeze the Jack server! It's NOT advisable to do major calculations here - shift those to another thread.
