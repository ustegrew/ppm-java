25 /**
26  * Emits a constant stream of {@link IEvented#gkEventTimerTick} events,
27  * on event per given interval. Modules can subscribe to this timer and
28  * will then receive the events. Recipients can use the timer events
29  * to perform regular tasks.
30  *
31  * Modules, in turn can send {@link IEvented#gkEventTimerAdjustInterval}
32  * events to change the interval time in this timer. This is the basis
33  * of the data contention compensation we use in the display engine.
34  *
35  * @author Peter Hoppe
36  * @see TController#Create_Connection_Events(String, String)
37  * @see TNodePump#_Resolve_DataLoss()
38  */

41  implements IControllable, IEvented
