Detailed Description

A generic map that stores objects by key and by index.

Element order is guaranteed during the lifetime of the map, elements are ordered in the sequence they are added. The index is zero based.
This store is not thread safe. Clients that need thread safety need to add their own thread safety mechanism.

Author
Peter Hoppe
