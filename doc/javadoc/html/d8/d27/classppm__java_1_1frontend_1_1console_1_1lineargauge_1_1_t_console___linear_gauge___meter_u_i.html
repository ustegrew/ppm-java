Detailed Description

The meter UI for a single gauge.

This class doesn't do any rendering, but it creates the meter bar string. The containing UI can retrieve the string (GetLevelBar()) and render it.

Peter Hoppe

String fColorCodeReset
private

Color code for resetting font to normal.

If there's no clipping or warning, this will be empty. We always draw a meter and then reset the font, so another meter bar can be drawn with normal color (e.g. one bar drawn at clip level, and the other bar at normal level.
