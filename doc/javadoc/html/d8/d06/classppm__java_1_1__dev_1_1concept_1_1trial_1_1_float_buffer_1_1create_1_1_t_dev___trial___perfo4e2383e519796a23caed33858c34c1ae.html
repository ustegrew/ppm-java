Detailed Description

Performance trial: How long does it take to create a new FloatBuffer? I wanted to see what performance impact the creation of a new FloatBuffer has.

Results above. Motivated by observation that JJack creates a new FloatBuffer with every iteration of the process() loop. From the results I concluded that the impact is negligible; therefore there's no problem with creating a new FloatBuffer with every iteration of the audio processing loop.
